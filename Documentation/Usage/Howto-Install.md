# Install TipTrack

TipTrack targets FreeCAD 1.0 or newer.

For development, clone the repository into FreeCAD's `Mod` directory and restart
FreeCAD:

```bash
git clone https://github.com/Rchiemstra/Timeline.git TipTrack
```

After publication, install TipTrack from FreeCAD's Addon Manager.
